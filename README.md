# 🎵 Vaishnavi Tunes

Vaishnavi Tunes is a modern and stylish music streaming app UI that allows users to explore, search, and enjoy their favorite songs. The app focuses on a smooth user experience with a visually appealing interface.

---

## 📱 App Preview

- 🔍 Search songs, artists, and tracks
- 🎧 Browse trending Bollywood hits
- 🇮🇳 Discover new Indian releases
- ❤️ Save songs to favorites
- 🏠 Simple navigation (Home, Search, Favorites)


2. Open the project folder:
